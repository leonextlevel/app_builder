# App Builder

Aplicativo para controle do dia a dia de treino.

## Pré-requisitos
Instale o Quasar CLI na máquina.

## Instalar as dependências
```bash
npm install
```

### Executar em modo dev
```bash
quasar dev -m pwa
```

### Executar o linter
```bash
npm run lint
```

### Build para produção
```bash
quasar build
```
